### 项目名称

- Photo Prompt - AI 图片展示

### 项目版本

- V1.2
- 完整功能版

### 功能描述

- 一个展示文生图信息和图片的网站
- 支持上传 ComfyUI 生成的 PNG 图片，自动读取工作流信息

### 核心功能

**图片展示**
- 显示多张精美的文生图卡片（9:16 比例）
- 卡片设计精美，有阴影和悬浮效果
- 点击图片可全屏预览

**图片上传**
- 点击卡片左上角的上传按钮上传图片
- 支持一次选择多张图片批量上传
- 支持 ComfyUI 生成的 PNG 格式
- 自动解析图片中的工作流信息（使用第一张图片的元数据）

**信息解析（从 ComfyUI PNG 读取）**
- 生图引擎：从 KSampler 节点提取采样器和调度器信息
- 模型：从 CheckpointLoaderSimple / UnetLoader 节点提取
- 正向提示词：从 CLIPTextEncode / CR Prompt Text 等节点提取
- LORA：从 LoraLoader / CR LoRA Stack 等 LORA 相关节点提取
- 自动过滤无效的 LORA 名称（如纯数字强度值）

**分类标签**
- 预置标签：人像、风景、抽象、场景、角色
- 支持自定义标签（保存在浏览器本地）
- 顶部标签栏可按标签筛选卡片
- 图片右上角标签点击可切换分类

**提示词操作**
- 提示词超过6行自动折叠，点击展开/收起
- 一键复制提示词到剪贴板

**多图轮播**
- 卡片支持多张图片轮播展示
- 自动播放：每3秒自动切换到下一张
- 左右切换按钮：鼠标悬停时显示，支持点击切换
- 位置指示器：右下角显示当前是第几张（如 "1/5"）
- 点击圆点指示器可快速跳转到指定图片
- 暂停/播放按钮：右上角控制自动轮播的启停
- 导出 GIF 按钮：可将轮播内容导出为 GIF 动图并自动下载

### 技术要求

- 所有代码放在一个HTML文件里，名字叫 text2img-index.html
- 纯前端实现，无需后端
- 无外部依赖库

### UI 风格

- 艺术摄影风格
- 主色调：黑白灰
- 卡片式设计，有阴影和悬浮效果
- 手机电脑都能正常显示（响应式布局）
